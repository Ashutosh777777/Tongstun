# SpeakUp

AI-powered voice conversation practice app.

## Services
- `frontend/` — React + Vite (UI, 3D scenes, voice capture)
- `middleware/` — Node.js + Express (API gateway)
- `api/` — FastAPI (voice pipeline, LLM, STT/TTS)
- `core/` — Django (auth, data, admin)

## Setup
See each service folder for individual setup instructions.
